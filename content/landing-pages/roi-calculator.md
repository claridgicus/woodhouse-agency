# AI Agent ROI Model

Use this framework to estimate the value of an AI agent deployment for your team.

## Step 2: Estimate agent coverage

Based on our deployments, a well-scoped agent layer typically handles:

- **60–80%** of routine tasks autonomously
- **15–25%** with human review/approval
- **5–15%** escalated fully to humans

Use **65%** as a conservative autonomous coverage rate.

## Step 3: Calculate the value

```
Annual automatable cost = employees × annual cost × % automatable time
Agent value             = annual automatable cost × 65% coverage
Implementation cost     = typically £50k–£200k depending on complexity
Payback period          = implementation cost ÷ (agent value ÷ 12) months
```

## Example

A finance team of 8 analysts (£70k each, fully loaded £95k) spending 60% of their time on routine reporting and reconciliation:

- Annual automatable cost: 8 × £95k × 60% = **£456k**
- Agent value at 65% coverage: **£296k/year**
- Implementation cost: **£120k**
- Payback period: **4.9 months**

After payback, the ongoing net benefit is approximately **£250k/year**.
